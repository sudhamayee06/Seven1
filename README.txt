How to use the programming environment
-------------------------------------------------------------------------
a) The workspace provided with the Web IDE has all project files already available in folder structure.
b) Write your code in index.html, script.js and style.css.
c) Click on LAUNCH APP to see the output from index.html, script.js and style.css.
d) In case if you are creating new JS file (like: example.js), Please load this JS file in index.html using the loadjscssfile function. It can be in any order.
